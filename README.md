# Sports-Performance-Analytics
Sports Performance Analytics using Power BI is a data visualization project that analyzes cricket match and player statistics to uncover performance insights. The dataset was imported into Power BI and cleaned using Power Query Editor, where columns were split and renamed, missing values were handled, and proper date and numeric formatting was ensured. Additional custom columns and DAX measures were created to enhance analysis. The project culminates in an interactive Power BI dashboard that visualizes player performance, match outcomes, and key statistics to provide meaningful sports analytics insights.
